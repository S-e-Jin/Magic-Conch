# 마법의 소라고동

이 프로젝트는 '네모바지 스폰지밥' 시즌 3 에피소드 42회에서 등장하는 '마법의 소라고동' 장난감을 모방하여,<br>
질문에 대해 랜덤한 답변을 제공하는 기능을 구현합니다.
<br>

## 주요 기능
- 사용자가 입력한 질문에 대해 랜덤한 답변을 제공
- 간단한 UI를 통해 질문하고 답변을 받을 수 있는 인터페이스를 제공
