# 마법의 소라고동

이 프로젝트는 '네모바지 스폰지밥' 시즌 3 에피소드 42회에서 등장하는 '마법의 소라고동' 장난감을 모방하여,<br>
사용자가 질문을 입력하면, 그에 대한 답변을 랜덤하게 제공하는 기능을 구현합니다.
<br>

## 주요 기능
- 사용자가 질문을 생각하고, 아이콘을 클릭하면 랜덤한 답변을 제공하는 인터랙티브한 경험 제공
- 직관적이고 간단한 UI를 통해 손쉽게 질문하고 답변을 받을 수 있는 인터페이스 제공
